# Proyecto de Machine Learning: Pricing

## Estructura de carpetas

- `data/raw/`: Datos originales sin procesar
- `data/processed/`: Datos procesados y listos para análisis/modelado
- `notebooks/`: Jupyter Notebooks para exploración y experimentos
- `models/`: Modelos entrenados y artefactos
- `app/`: Aplicación Streamlit
- `docs/`: Documentación del proyecto

## Requisitos
Ver `requirements.txt` para dependencias.

## Uso
1. Coloca los datos en `data/raw/`.
2. Usa los notebooks en `notebooks/` para análisis y modelado.
3. Guarda modelos en `models/`.
4. Desarrolla la app en `app/`.
